# JLPT N4 한자읽기 학습 앱 🇯🇵

> 다락원 「JLPT 한권으로 끝내기 N4」 한자읽기 파트 기반의 React 학습 앱

---

## 📚 학습 내용

| 과 | 기간 | 어휘 수 | 확인문제 |
|---|---|---|---|
| 1과 | 기출 2016~2024 | 96개 | 6세트 (60문항) |
| 2과 | 기출 2000~2015 | 108개 | 11세트 (110문항) |
| 개별 한자 | 1·2과 공통 | 96자 | — |

---

## ✨ 주요 기능

### 🈳 개별 한자 학습 (신규)
- 1·2과에서 사용된 핵심 한자 96자를 개별 학습
- **앞면**: 한자 대형 표시 + **모양 힌트 버튼** (예: 火 — 장작이 포개져 있고 불씨가 위로 솟아오르는 모습)
- **뒷면**: 훈독/음독, 한국어 뜻, 예시 단어
- 1과/2과/전체 필터 지원

### 📚 어휘 학습
- 플래시카드 방식 (앞면 한자 → 뒤집으면 히라가나+뜻)
- **앞면에서도 암기 힌트 버튼 사용 가능** (신규)
- ⭐ 중요 어휘 필터 / ⚠️ 오답 필터

### 🧪 퀴즈 3종
- ① 히라가나 → 한국어
- ② 한자 → 한국어  
- ③ 히라가나 → 한자

### 📝 실전문제풀이
- 기출 확인문제 17세트 전체 수록
- 후리가나(ふりがな) 표시
- **정답 후 문장 한국어 해석 표시** (신규)
- 오답 자동 저장

### ⚠️ 오답 복습
- 어휘/문제 오답 통합 관리
- 체크 버튼으로 오답 해제

## 🚀 실행 방법

### Claude.ai에서 바로 사용
1. [claude.ai](https://claude.ai) 접속
2. `App.jsx` 파일 내용을 붙여넣고 **React Artifact**로 실행

### 로컬 개발 환경
```bash
# Vite + React 프로젝트 생성
npm create vite@latest jlpt-n4 -- --template react
cd jlpt-n4

# App.jsx 교체
cp App.jsx src/App.jsx

# 실행
npm install
npm run dev
```

> ⚠️ Tailwind CSS 불필요 — 모든 스타일이 CSS-in-JS로 내장되어 있습니다.

## 📁 파일 구조

```
src/
└── App.jsx          # 전체 앱 (단일 파일)
    ├── CSS           # 다크 테마 인라인 스타일
    ├── KANJI_DB      # 개별 한자 데이터 (96자)
    ├── V1 / V2       # 1·2과 어휘 데이터
    ├── P1 / P2       # 1·2과 확인문제 데이터 (한국어 해석 포함)
    └── Components
        ├── App              # 루트 (화면 상태 관리)
        ├── HomeScreen       # 홈 화면
        ├── MenuScreen       # 과별 메뉴
        ├── KanjiScreen      # 개별 한자 학습 ✨
        ├── VocabScreen      # 어휘 플래시카드
        ├── QuizScreen       # 퀴즈 3종
        ├── PracticeSetScreen   # 실전문제 목록
        ├── PracticeSolveScreen # 실전문제 풀이 ✨
        └── WrongScreen      # 오답 복습
```

## 📖 데이터 형식

```js
// 어휘: [한자, 히라가나, 한국어, 힌트, 중요여부]
['青い', 'あおい', '파랗다', '하늘(日)과 달(月)이 빚어내는 파란빛 🌊', true]

// 한자: [한자, 음독, 훈독, 한국어뜻, 모양힌트, 예시단어, 과]
['火', 'か', 'ひ', '불', '장작이 포개져 있고 불씨가 위로 솟아오르는 모습 🔥', '火事(かじ)화재', '2']

// 문제: [후리가나문장, 밑줄단어, [선택지], 정답인덱스, 중요여부, 한국어해석]
['{田中|たなか}さんのお姉さんは...', 'お姉さん', ['おねさん',...], 2, true, '다나카 씨의 언니는...']
```
